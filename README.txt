
# Запуск
1. Убедитесь, что у вас установлен Python.
2. Запустите `run.bat` (или `start_bot.bat`).

# Настройка
1. Откройте файл `.env`.
2. Впишите ID главного администратора (ваш ID) в `ADMIN_IDS`. Например: `ADMIN_IDS=123456789,987654321`.
3. Убедитесь, что бот добавлен в канал `-1003893023453` как администратор (с правом создавать пригласительные ссылки).

# Функционал
- **/start**: Запуск бота пользователем.
- **/admin**: Админ-панель (доступна только админам).
- **Оплата**: Сейчас работает в тестовом режиме (кнопка "Тест"). Генерирует одноразовую ссылку на канал.
